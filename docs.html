What is AmfreOS?

AmfreOS is a Debian-based desktop Linux distribution optimized for stability and daily use. AmfreOS offers simplified package and system management commands to make Linux usage easier; it avoids unnecessary complexity while providing control to the user.

Created by a curious IT enthusiast from Western Kenya, AmfreOS is designed with consideration for users in Kenya and across Africa, focusing on reliability, ease of use, and efficient performance even on modest hardware.

Installation

1. Download the ISO file from the website. Copy it to your USB drive using Ventoy or Rufus.

2. Enter your motherboard's BIOS and boot your system from the USB drive.

Note: AmfreOS supports most modern hardware. If your GPU is Nvidia, ensure it is compatible with nouveau.

AmfreOS may not boot on very old hardware. A minimum of 4GB RAM is recommended. Additionally, problems may arise on very new hardware with low Linux compatibility.

Once you reach the Boot Screen, you can start in Live mode to try AmfreOS, or start the Debian Installer to install it to your hard drive. Complete the necessary steps via the GUI, install your system, and start exploring!

Note for Live Mode: When you press shut down in Live mode, you may see the Plymouth screen. Simply press the Enter key; this will safely remove the live-medium and resolve the shutdown issue.

AmfreOS Commands

No more getting lost searching for repos on the internet or risking your system with old or fake repos. AmfreOS saves you from wasting hours with its special commands. You can download the software you want using up-to-date and official repos. At the same time, system management scripts will save you from getting lost in the terminal for hours!

System Management

To update the system quickly, easily, and safely:

sudo amfre-update

To easily change your DNS:

sudo amfre-change-dns

App Installation Shortcuts

Install Flatpak quickly:

amfre-install-flatpak

Download Steam easily:

amfre-install-steam

Download Brave Browser easily:

amfre-install-brave

Download VSCode easily:

amfre-install-vscode

Download AI IDEs (Cursor, Windsurf, AntiGravity available!):

amfre-install-vibecoding

Download Office suites easily:

amfre-install-officesuite

Download Video Editing and Design programs:

amfre-install-editor

Download Music Production and Audio software:

amfre-install-producer

Performance & Utility

Install the fast Liquorix Kernel:

amfre-install-liquorix

*Check compatibility with your Nvidia card.

Use Zram to increase performance if you have low RAM:

amfre-zram-settings

Run Local AI models:

amfre-install-ollama

Get detailed network information:

amfre-net-info

Get system information:

amfre-fetch

Kenyan Focus

AmfreOS is developed in Kenya with consideration for local computing needs:

  • Network Optimization: Optimized for East African network conditions
  • Localized Support: Documentation and support considering local context
  • Hardware Considerations: Tested and optimized for hardware commonly available in Kenya
  • Open Source Advocacy: Promoting open source software adoption in Kenya
  • Community Driven: Developed with input from Kenyan users and IT professionals

AmfreOS aims to provide a stable, reliable Linux distribution that works well in Kenyan computing environments while being accessible to users worldwide.
