AmfreOS

Linux that just works

AmfreOS is a Debian (stable)–based Linux distribution designed as a reliable daily driver. It focuses on simplicity, stability, and privacy, while remaining easy to use for everyday work, learning, and development.

About AmfreOS

AmfreOS is a Debian-based GNU/Linux distribution built for people who want an operating system that works out of the box. It is suitable for daily use, including browsing, development, office work, learning, and general computing.

In addition to being a normal desktop OS, AmfreOS includes optional anti-forensic and privacy tools. These tools help reduce leftover system traces such as logs, swap data, and temporary files, without interfering with normal usage.

FAQ

+

Q: What is AmfreOS?

A: AmfreOS is a Debian-based daily-driver Linux distribution focused on stability, usability, and privacy.

Q: Is AmfreOS suitable for beginners?

A: Yes. It works out of the box and follows standard Debian workflows.

Q: Does AmfreOS include anti-forensic tools?

A: Yes, optional tools are included to clean logs, wipe swap, and reduce system traces.

Q: Is AmfreOS open source?

A: Yes. It is built entirely on open-source software.

Installation

+
  • Download the AmfreOS ISO
  • Flash to USB using Ventoy or Rufus
  • Boot into live mode or install via Calamares

Usage

+

AmfreOS behaves like a normal Linux desktop. Privacy tools can be enabled or disabled depending on user preference.
