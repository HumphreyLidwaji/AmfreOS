Daily Driver Linux with Anti-Forensic Tools

AmfreOS is a Debian-based Linux distribution focused on stability, usability, privacy, and forensic security. It can run live or be installed via Calamares.

About AmfreOS

AmfreOS is a Debian-based Linux daily-driver designed for everyday usage, stability, and enhanced privacy. It comes with optional anti-forensic tools that can remove traces, wipe swap, and logs automatically.

Q: What is AmfreOS?

A: AmfreOS is a Debian-based daily-driver Linux distribution focused on stability, usability, and privacy.

Q: Is AmfreOS suitable for beginners?

A: Yes. It works out of the box and follows standard Debian workflows.

Q: Does AmfreOS include anti-forensic tools?

A: Yes, optional tools are included to clean logs, wipe swap, and reduce system traces.

Q: Is AmfreOS open source?

A: Yes. It is built entirely on open-source software.
